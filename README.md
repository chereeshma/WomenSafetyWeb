# Women Safety Emergency Alert System

The Women Safety Emergency Alert System is a web-based safety platform developed using Flask that helps users quickly send emergency alerts along with their live location to trusted contacts. The application integrates communication services and data visualization to improve emergency response and safety awareness.

## Features

* Emergency alert button to send instant notifications
* SMS, WhatsApp, and Email alerts using Twilio API
* Real-time location tracking
* Interactive map visualization using Folium
* Dashboard displaying crime-prone or high-risk zones
* User alert data storage using SQLite database

## Technologies Used

* Python
* Flask
* SQLite
* Twilio API
* Folium
* HTML, CSS, JavaScript

## Purpose

The system is designed to enhance personal safety by enabling fast communication during emergencies and by providing crime zone insights using data analytics.
